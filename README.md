# Trader Behavior vs Bitcoin Market Sentiment

This project analyzes how trader behavior (volume, leverage, profitability) changes across
Bitcoin market sentiment regimes (Fear vs Greed) using:

- Hyperliquid historical trader data
- Bitcoin Fear & Greed index

## Structure

- `notebook_1.ipynb`: Main Colab notebook with EDA, feature engineering, and analysis.
- `csv_files/`: Aggregated datasets generated from the notebook.
- `outputs/`: PNG charts used in the report.
- `ds_report.pdf`: Final report with methodology and key insights.

## How to run

1. Open `notebook_1.ipynb` in Google Colab.
2. Upload `historical_data.csv` and `fear_greed_index.csv`, or mount Google Drive.
3. Run all cells.
4. Generated CSVs will appear in `csv_files/` and plots in `outputs/`.
